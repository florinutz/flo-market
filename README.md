## Installation

### From marketplace

```
/plugin marketplace add florinutz/flo-marketplace
/plugin install narra@flo-marketplace
```

### Manual (single plugin)

```bash
git clone https://github.com/florinutz/flo-market.git /tmp/claude-plugins
cp -R /tmp/claude-plugins/plugins/narra ~/.claude/plugins/narra
```
